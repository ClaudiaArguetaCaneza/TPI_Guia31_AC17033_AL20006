CONCLUSIÓN

Luego de la realización de la guía 31 enfocada en el manejo de XML, JSON y CSS, los cuales son lenguajes 3 de programación utilizados para la representación, almacenamiento y organización de datos, que favorecen el desarrollo de aplicaciones web y móviles eficientes.

Respecto al lenguaje XML (Extensible Markup Language) se concluye que es un lenguaje de marcado utilizado para representar datos estructurados. Con la peculiaridad de que le da completa libertad al programador al momento de elegir el conjunto de etiquetas a utilizar en el contenido y la estructura de los datos.

En el caso del uso de archivos JSON (JavaScript Object Notation) estos son parte de un formato de intercambio de datos basado en los objetos JavaScript descritos ampliamente en la realización de la guía 41c. en esta ocasión se convirtió los archivos XML a formato JSON obteniendo objetos escritos en un lenguaje ligero y los cuales fueron probados en la consola JavaScript.

Para finalizar la práctica se usó el lenguaje CSS (Cascading Style Sheets) para establecer el estilo de los elementos de la página web de los ejercicios XML realizados. En donde se concluye que CSS es una gran herramienta a la hora de definir el formato estándar de nuestros sitios web, seleccionando la fuente, tamaño, colores, entre otros aspectos relacionados de los elementos.
